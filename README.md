# Password Generator Web App

## 📌 Project Description

This project is a simple **Password Generator Web Application** built using **Python** and **Streamlit**.
It helps users generate secure and random passwords instantly.

The application allows users to choose the password length and generates a strong password using random characters.

---

## 🚀 Features

* Generate secure random passwords
* User can select password length
* Simple and interactive user interface
* Fast password generation

---

## 🛠️ Technologies Used

* Python
* Streamlit

---

## ▶️ How to Run the Project

1. Install Streamlit

```
pip install streamlit
```

2. Run the application

```
streamlit run password_generator_app.py
```

3. The app will open in your browser automatically.
